# 一、核心技术栈掌握情况

## 1. 前端基础

- **HTML5/CSS3**：掌握语义化标签、Flex/Grid布局、响应式设计、CSS动画（），能还原常见UI设计稿；
- **JavaScript ES6+**：熟练运用解构、箭头函数、Promise/async-await、模块化，理解事件循环、闭包、原型链等核心概念；
- **TypeScript**：基础类型定义、接口、泛型，能配合Vue3进行类型约束。

## 2. Vue3生态

- **基础语法**：响应式原理(ref/reactive)、Composition API、计算属性、侦听器、条件/列表渲染；
- **工程化开发**：Vue Router路由配置（懒加载、导航守卫）、Pinia状态管理、Vite构建工具配置；
- **组件化**：组件通信(props/emit、依赖注入)、插槽、组件封装与复用；
- **生命周期**：onMounted/onUnmounted等组合式API生命周期运用。

## 3. React生态（需要强化）

- **基础**：函数组件、Hooks(useState/useEffect/useContext)、组件通信；
- **工程化**：Create React App/Vite+React、React Router基础配置。

## 4. UI组件库

- **Element Plus**：表格、表单、对话框、菜单等复杂组件，二次封装业务组件；
- **Ant Design**：基础组件使用，了解其设计规范；
- **Tailwind CSS**：实用类名开发、自定义配置、响应式工具类。（）

## 5. 前后端交互（需要强化）

- **Axios**：全局实例配置、拦截器封装(请求token注入、响应统一处理)、错误码处理(401/403/500)；
- **RESTful接口**：接口联调、跨域处理、请求优化(防抖节流、重复请求取消)；
- **Mock数据**：本地JSON模拟、接口数据格式转换。

## 6. 工程化与工具

- **Git**：分支管理、代码合并、冲突解决、commit规范；
- **包管理**：npm/pnpm/yarn，依赖版本管理；
- **代码规范**：ESLint+Prettier配置、Husky代码提交前检查；
- **构建工具**：Vite配置优化、Webpack基础配置理解。
