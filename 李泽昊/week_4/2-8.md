## 一，Redux
### 1.1 Redux是什么
Redux是一个JavaScript状态管理库，专门用于管理应用的全局状态
### 1.2 Redux特点
- 单一数据源，将整个应用中的状态保存在一个store中
- 状态只读，唯一修改state的方式为dispatch一个action
- 纯函数修改：使用纯函数reducer来执行状态修改
### 1.3 Redux的应用场景
- 组件层级深，多个层级的组件需要共享状态的时候可用Redux统一管理state
- 跨层级通信，非父子关系的组件需要通信
- 状态需要持久化的时候可用Redux，或者状态更新逻辑复杂，可用Redux管理
## Redux的核心概念
### 1.store
store是Redux的核心对象，保存应用的整个状态树
### 2.action
action是一个普通的对象用于描述发生了什么
```javascript
// Action 的基本结构
const action = {
    type: 'ADD_TODO',      // 必须有 type 字段
    payload: {             // 可选的数据字段
        text: 'Learn Redux'
    }
};
```
### 3.reducer
reducer是一个纯函数,接受旧的state和action,返回新的state
```javascript
function reducer(state, action) {
    switch (action.type) {
        case 'ADD_TODO':
            return [...state, action.payload];
        default:
            return state;
    }
}
```
### 4.dispatch
dispatch是一个方法,用于发送action到store
```javascript
store.dispatch({
    type: 'ADD_TODO',
    payload: { text: 'Learn Redux' }
});
```
### 5.selector
selector是一个函数，用于从store中获取特定的state
```javascript
const todos = store.getState().todos;
```
## 二，React-Redux
## Redux基础API
### 1.createStore
创建Redux store
```javascript
import { createStore } from 'redux';

// 创建 reducer
function counterReducer(state = { count: 0 }, action) {
    switch (action.type) {
        case 'INCREMENT':
            return { count: state.count + 1 };
        case 'DECREMENT':
            return { count: state.count - 1 };
        default:
            return state;
    }
}

// 创建 store
const store = createStore(counterReducer);
```
其中state为需要管理的状态变量,action为需要传入的行为对象,通过createStore方法将定义好的reducer函数构建成一个store对象
### 2.getState
获取当先的state
### 3.dispatch
dispatch方法用于派发action,注意action对象中必须要有type属性
### 4.subscribe
订阅state变化
```javascript
// 订阅
const unsubscribe = store.subscribe(() => {
    console.log('State changed:', store.getState());
});

// 取消订阅
unsubscribe();
```
### 5.replaceReducer
替换store的reducer

## react-redux库下的状态管理
react-redux是一个用于管理react组件状态的库，它将react组件与redux进行连接，使得组件可以通过props获取redux中的状态数据，并通过dispatch方法发送action来修改状态数据。
### 1.配置
react-redux提供了一个Provider核心组件,该组件接收一个store属性,并将其传给子组件
```javascript
// index.js
import React from 'react';
import ReactDOM from 'react-dom/client';
import { Provider } from 'react-redux';
import store from './store';
import App from './App';

const root = ReactDOM.createRoot(document.getElementById('root'));
root.render(
    <Provider store={store}>
        <App />
    </Provider>
);
```
该组件还可以嵌套使用
### react-redux中的hook函数
react-redux提供了两个核心的hook函数，useSelector和useDispatch，分别用于获取状态数据和发送action。
- useSelector: 用于获取状态数据
```javascript
import { useSelector } from 'react-redux';

function Counter() {
    // 选择整个 state
    const state = useSelector(state => state);
    
    // 选择特定字段
    const count = useSelector(state => state.counter.count);
    
    // 选择嵌套字段
    const user = useSelector(state => state.user.data);
    
    return <div>{count}</div>;
}
```
- 选择器函数
```javascript
// 简单选择器
const selectCount = state => state.counter.count;

// 参数化选择器
const selectTodoById = (state, todoId) => state.todos.items.find(t => t.id === todoId);

// 组合选择器
const selectCompletedTodos = state => state.todos.items.filter(t => t.completed);

// 使用
function Component() {
    const count = useSelector(selectCount);
    const todo = useSelector(state => selectTodoById(state, 1));
    const completedTodos = useSelector(selectCompletedTodos);
    
    return <div>...</div>;
}
```
- useDispatch: 用于发送action
```javascript
import { useDispatch } from 'react-redux';
import { increment, decrement } from './counterSlice';

function Counter() {
    const dispatch = useDispatch();
    
    const handleIncrement = () => {
        dispatch(increment());
    };
    
    const handleDecrement = () => {
        dispatch(decrement());
    };
    
    return (
        <div>
            <button onClick={handleIncrement}>+</button>
            <button onClick={handleDecrement}>-</button>
        </div>
    );
}
```
- 还可以派发代参数的action,以及异步action,thunk action
